## 📝行修补对象描述法

**%操作% %修饰%<%对象名称或内容%>:%补充%**，多层关系用`()`表示包含

- 用于Git提交信息

    | 情况     | 隔开符号         |
    | -------- | ---------------- |
    | 多个文件 | `,`<br/>前后紧贴 |
    | 不同操作 | `;`<br/>后位留空 |

    | 修饰符  | 值（一般结构）                            | 意义               |
    | ------- | ----------------------------------------- | ------------------ |
    | New     | %文件名称%                                | 新建               |
    | Update  | %文件名称%:%描述%（%关键词%<%对象名称%>） | 增加新特性         |
    | Rebuild | %文件名称%:%改变%（%关键词%<%对象名称%>） | 重写或修改已有特性 |
    | Fixed   | %文件名称%:%错误%（%关键词%<%对象名称%>） | 修复错误           |
    | Delete  | %文件名称%                                | 删除               |

    | 关键词   |
    | -------- |
    | var      |
    | function |
    | class    |
    | error    |
    | todo     |
    | param    |

	> 关键词补充：%关键词% => %关键词%:%补充%
	
	例子：
	
	```
	New example.txt; Update example.cpp:Rename (var:float)<varFloat> from (var:int)<varInt>
